Imprimir uma ontologia

O On_Tool possui sistema de impressão, para que você possa ter em papel as suas ontologia. Para imprimir uma ontologia, siga os seguintes passos:

  1. Ajuste o tamanho da imagem usando o zoom . Para saber como ajustar o zoom, clique aqui .

  2. Clique em Arquivo, na barra de menus. Em seguida, clique em “Imprimir”.

  3. Se seu computador tiver uma impressora instalada corretamente, surgirá um diálogo de impressão, com as devidas configurações para a impressora. Esse diálogo difere de acordo com o sistema operacional em execução. Se seu computador não tiver uma impressora instalada, surgirá uma mensagem de erro avisando sobre isso.

  4. Você também pode substituir o passo 2 clicando em  na barra de tarefas.
